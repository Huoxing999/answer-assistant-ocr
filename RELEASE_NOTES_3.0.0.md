# 答题参考助手 OCR 3.0.0

## 新增

- 自动识别题库中的题目列、答案列和 A/B/C/D 选项列。
- 设置窗口支持手动修正题目列、答案列和选项列。
- 答案为 `A`、`AB`、`ABCD` 时，自动显示对应选项内容。
- 结果窗口新增“识别题目”和“匹配题目”分区。
- 参考答案区域只显示最高匹配结果。
- OCR 后台预热，减少第一次识别等待。

## 优化

- 多选答案先显示答案字母，再显示选项内容。
- 调整参考答案卡片字号和布局。
- 提高屏幕变化检测阈值，减少重复 OCR。
- OCR 前增加灰度和自动对比度处理。

## 修复

- 修复重新设置题库后可能闪退的问题。
- 修复 OCR 错误选项覆盖题库选项的问题。
- 修复识别框提示文字乱码。
- 修复 exe 图标，使用 2.0 版本同款图标。

## 使用

下载 `启动答题助手3.0.exe` 后双击运行。

首次运行请选择题库文件，并确认题目列、答案列和选项列是否正确。
