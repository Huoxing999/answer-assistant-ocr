# 答题参考助手 OCR 3.0

基于 OCR 的桌面答题参考工具。程序可以框选屏幕题目区域，自动识别题目文字，匹配本地题库，并显示参考答案。

## 3.0 版本重点

- 新增题库列自动识别：自动判断题目列、答案列和 A/B/C/D 选项列。
- 支持手动修正列选择：自动识别不准时，可以在设置窗口里手动选择。
- 参考答案显示选项内容：例如答案为 `ABC` 时，会显示 `ABC`，并在下方显示 A/B/C 对应选项。
- 只显示最高匹配结果：参考答案区域只保留概率最高的一条，避免多个结果干扰。
- 识别题目和匹配题目分区显示：左侧上方显示 OCR 识别文本，下方显示题库匹配到的题目。
- OCR 后台预热：启动后后台加载 EasyOCR，减少第一次识别等待时间。
- 修复重新设置题库后闪退的问题。
- 修复 OCR 错误选项覆盖题库选项的问题：题库选项优先，OCR 只用于补缺失项。
- 修复选框提示文字乱码和 exe 图标问题。

## 下载使用

在 GitHub Releases 中下载最新版：

https://github.com/Huoxing999/answer-assistant-ocr/releases/latest

下载后双击：

```text
启动答题助手3.0.exe
```

首次运行会弹出设置窗口，选择题库文件并确认列设置。

## 题库格式

支持：

- `.xlsx`
- `.xls`
- `.csv`

推荐题库表头包含这些列：

| 列类型 | 示例表头 |
| --- | --- |
| 题目 | 题目、题干、Question |
| 答案 | 答案、正确答案、Answer |
| 选项 | A、B、C、D 或 选项A、选项B、选项C、选项D |

如果答案列里是 `A`、`ABCD` 这类字母答案，程序会自动补全对应选项内容。

示例：

```text
答案：ABC

A 汽车
B 火车
C 飞机
```

## 使用流程

1. 启动程序。
2. 在设置窗口选择题库文件。
3. 检查自动识别出来的题目列、答案列和选项列。
4. 如果自动识别不对，手动选择对应列。
5. 拖动绿色识别框覆盖题目区域。
6. 右键识别框，选择“锁定选框（开始识别）”。
7. 查看结果窗口中的识别题目、匹配题目和参考答案。

## 窗口说明

左侧结果窗口：

- 上方：OCR 识别到的题目
- 下方：题库中匹配到的题目

右侧结果窗口：

- 显示最高匹配结果
- 显示答案字母
- 显示答案对应的选项内容
- 自动复制最佳答案到剪贴板

## 右键菜单

识别框：

- 锁定选框（开始识别）
- 解锁选框
- 题库设置
- 退出

结果窗口：

- 字体大小
- 退出

## 配置项

| 配置项 | 当前默认值 | 说明 |
| --- | ---: | --- |
| `POLL_INTERVAL` | `0.8` | 锁定后识别轮询间隔，单位秒 |
| `CHANGE_THRESHOLD` | `8` | 屏幕变化检测阈值，越大越不容易重复触发 OCR |
| `MATCH_THRESHOLD` | `0.35` | 题目匹配相似度阈值 |
| `MAX_RESULTS` | `3` | 内部最大匹配候选数，界面只显示最高匹配 |
| `FONT_SIZE` | `25` | 默认字体大小 |

## 常见问题

### 参考答案选项内容不对

优先检查设置窗口中的 A/B/C/D 选项列是否选择正确。3.0 会优先使用题库选项列，OCR 只用于补缺失的选项。

### 第一次识别较慢

EasyOCR 首次加载模型需要一些时间。3.0 已加入后台预热，启动后稍等一会儿再开始识别会更流畅。

### 识别不到题目

- 确认识别框完整覆盖题目文字。
- 尽量避免把其它窗口文字、菜单、弹窗框进识别区域。
- 可以放大题目区域或调整字体大小后再识别。

### 图标没有立即刷新

Windows 资源管理器可能会缓存 exe 图标。按 F5 刷新文件夹，或重启资源管理器后会更新。
